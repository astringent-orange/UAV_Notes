购买便携路由器以便固定ip与外出携带

首先登陆路由器 192.168.1.1
设置管理员密码 mig12345678

其次更改路由模式为局部，从而不需要连接外部网络。采用192.168.8.x频段，此时路由器地址192.168.8.1

仅保留5G频段，并关闭密码
2.4G频段与无人机控制信号频段重叠可能导致干扰

设置静态ip
笔记本 ubuntu 192.168.8.50
nuc13 192.168.8.101

命令行模式连接wifi并设置静态ip方法
```bash
# 其中con-name为别名 ssid为WiFi名称  address 为静态ip
sudo nmcli connection add type wifi ifname wlo1 con-name GL-MT3000-5G \
  ssid "GL-MT3000-5G" \
  ipv4.method manual \
  ipv4.addresses 192.168.8.101/24 \
  ipv4.gateway 192.168.8.1
  
sudo nmcli connection up GL-MT3000-5G
```
