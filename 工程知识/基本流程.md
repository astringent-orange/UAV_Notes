
### 查看mavros连接状态

**端口1**
`roscore`
启动ros

**端口2**
`roslaunch mavros px4.launch fcu_url:=/dev/ttyACM0:57600`
启动mavros并指定端口

**端口3**
`rostopic list`
查看话题

`rostopic echo -n 5 /mavros/state`
查看连接状况
```
header: ...
connected: True
armed: False
guided: False
mode: "MANUAL"
...
```
关注其中connected的值


### 手动检查IMU
`watch -n 1 "rostopic echo -n 1 /mavros/imu/data"`
观察其中orientation信息
