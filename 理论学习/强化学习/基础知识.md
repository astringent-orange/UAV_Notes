### 问题表示
强化学习通常用 **马尔可夫决策过程（MDP, Markov Decision Process）** 来建模。一个 MDP 用五元组表示：
(S,A,P, R,γ)

| 符号             | 名称                             | 含义                              |
| -------------- | ------------------------------ | ------------------------------- |
| **S**          | 状态空间 (State space)             | 环境中所有可能的状态集合                    |
| **A**          | 动作空间 (Action space)            | 智能体在每个状态下可执行的动作集合               |
| **P(s'\|s,a)** | 状态转移概率(Transition probability) | 执行动作 a 后，从状态 s 转移到 s′ 的概率分布     |
| **R(s,a)**     | 奖励函数 (Reward function)         | 执行动作 a 后获得的即时奖励                 |
| **γ**          | 折扣因子 (Discount factor)         | 控制未来奖励的重要程度，0<γ≤10 < γ ≤ 10<γ≤1 |
